# github serarch project skill 

## 1. 搜索仓库标题、描述和README 

- `in:name 关键词`

- `in:description 关键词`

- `in:readme 关键词`

## 2. start 和 fork 数 

- `stars:>数字` 或 `stars:10..20`

- `fork:>数字` 或 `fork:10..20`

## 3. 仓库大小 

- `size:>= 5000`

## 4. 仓库是否在维护 

- `pushed:>2021-01-23` 

## 5. 仓库的使用语言 

- `language:java`
